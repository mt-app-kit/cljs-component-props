component-props.api
tutorial
Dynamic properties
plain
Dynamic properties of components are stored in a state atom, updated by functions
and should be merged onto the static properties of the component.
tutorial
Property presets
plain
Property presets of components are registered reusable preset maps or functions
applied on the property map of the component.

apply-presets
(defn apply-presets
  [{:keys [presets] :as component-props}]
  (letfn [(f0 [component-props preset] (-> component-props (f1 preset) (f2 preset)))
          (f1 [component-props preset] (apply-preset  component-props preset))
          (f2 [component-props preset] (apply-presets component-props))]
         (if (-> presets vector? not)
             (-> component-props)
             (reduce f0 (dissoc component-props :presets) presets))))
description
Applies its presets (if any) on the given component property map.
